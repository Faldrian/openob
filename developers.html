Developer Info¶

OpenOB’s entry point for standalone applications is the OpenOB class. This sets up logging and configuration objects.

class openob.openob.OpenOB[source]¶

The base OpenOB class

Provides basic interfaces onto shared objects and manages logging setup

class openob.link_manager.LinkManager[source]¶

OpenOB Manager. Handles management of links, mostly recovery from failures.

class openob.link_config.LinkConfig(link_name)[source]¶

OpenOB LinkConfig object. Abstracts all the relevant configuration parameters for an OpenOB link, setting/retrieving etc.

get(key, block=False)[source]¶

Get a configuration value

Can block on a non-existent key if desired with the block kwarg; in this mode the get will block until the key has a value (that is not False)

Set a configuration value
